# QUEST - API GitHub

## Descrição
Esse projeto tem como objetivo buscar as seguintes informações de usuário do GutHub:
- Nome completo
- Login
- Bio
- Seguidores
- Seguindo
- Repositórios, incluindo quantidade de forks, estrelas, watchers e linguagem de programação utilizada
- Eventos

## Linguagens
- HTML
- CSS
- JavaScript
